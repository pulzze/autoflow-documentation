# switch

## Summary

This Action returns values that fits on switch case.<br/>
If the input value is not in the case list, it returns the value of default.

## Description

The <span class="action-field">SWITCH</span> ACTION evaluates an expression, matching the expression's value against a series of <span class="action-field">*CASE*</span> clauses,
and executes statements after the first <span class="action-field">*CASE*</span> clause with a matching value, until <span class="action-field">*DEFAULT*</span> statement is encountered. <span class="action-field">*DEFAULT*</span> clause of a switch statement will be jumped to if no <span class="action-field">*CASE*</span>  matches the expression's value.

## Fields

In <b>*SWITCH Action*</b> there are 5 type fields.

- <span class="action-field">SWITCH</span> : Criteria for SWITCH
- <span class="action-field">CASE</span> : Target's CASE
- <span class="action-field">Return</span> : Return result IF the Creteris is 'CASE'
- <span class="action-field">Add CASE</span> : Add more CASE
- <span class="action-field">Default Return</span> : Default return result

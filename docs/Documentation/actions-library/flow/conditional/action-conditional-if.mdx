# if

## Summary

This Action returns values that fits the condition.

## Description

The <span class="action-field">*IF*</span> statement executes a statement if a specified condition is truthy. <br/>
If the condition is false, another statement in the optional <span class="action-field">*ELSE-IF*</span> clause will be executed.

## Fields

In <b>*IF Action*</b> there are 3 type fields.
- <span class="action-field">IF</span> : To check condition
- <span class="action-field">return</span> : Return result If the condition is True
- <span class="action-field">ELSE-IF</span> : Add more IF condition

<img src={if01} alt="Version Location" class="myResponsiveImg" width="800px"/>

### - <span class="action-field">IF</span>
This field is to check If condiion. <br/>
Basically, default is 'Boolean Type'. <br/>
Also, you can change the type to <b>*'References Data', 'Action', 'Condtions', 'Comparisons'*</b> like below. 

<img src={if02} alt="Version Location" class="myResponsiveImg" width="500px"/>

### - <span class="action-field">return</span>

This field determines how to return the value that returns when the if statement is true. <br/>
You can change the type to <b>*'Values (string, number...)', 'References Data', 'Action' ...'*</b> like below. 

<img src={if03} alt="Version Location" class="myResponsiveImg" width="500px"/>


### - <span class="action-field">Add ELSE-IF</span>

This field to use when you want to add more if statements. <br/>
Click to add more else-if statements, as shown below.

<img src={if04} alt="Version Location" class="myResponsiveImg" width="800px"/>

:::tip
If you add Else-if and set the boolean true value in <span class="action-field">ELSE-IF</span> field,<br/>
the return value can be set to a value that comes out as default in the if statement.
:::

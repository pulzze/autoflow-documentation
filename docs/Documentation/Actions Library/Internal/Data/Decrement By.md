# Decrement By

Action for decrementing a value by a given value.

&gt; **Properties**
&gt; 

**Value**:                     Value to decrement

**Amount**:                 Integer amount to decrement by

&gt; **Output**
&gt; 

**Output-location**: Write to location

&gt; **Example**
&gt; 

If the value is 10 and amount is 1, the output is 9. (10-1=9)

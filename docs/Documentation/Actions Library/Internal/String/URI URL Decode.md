# URI URL Decode

String URI URL Decode. Decodes string as “x-www-form-urlencoded”.

&gt; **Properties**
&gt; 

**String**:                     Location of the string to decode

&gt; **Output**
&gt; 

**Output-location**: Location to store the output data

**Example**

URL of “%3Call+in%2F” will return “&lt;all in/”

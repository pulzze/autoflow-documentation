# URI Percent Decode

URI Percentage Decode.

&gt; **Properties**
&gt; 

**String**:                     Location of the URI to decode

&gt; **Output**
&gt; 

**Output-location**: Location to store the output data

**Example**

Decoding to the URI “https%3A%2F%2Finteractor.com” will return “https://interactor.com”

# URI URL Encode

String URI URL Encode. Encodes string as “x-www-form-urlencoded”.

&gt; **Properties**
&gt; 

**String**:                     Location of the string to encode

&gt; **Output**
&gt; 

**Output-location**: Location to store the output data

**Example**

URL of “put: it+й” will return ” put%3A+it%2B%D0%B9″

# N to 1

Received data from multiple sources in a different format and normalized the data into a standardized data format.


## Data Objective

Duplicate data is a common problem that data normalization needs to solve.  In this example, user data in different formats is normalized into a standardized JSON format.

<img src={n_to_1_example} alt="N to 1 example" class="myResponsiveImg" width="800px"/>

&nbsp;<br />

:::info Process
1. The process will be to use <a href="/docs/Documentation/actions-library/flow/conditional/action-conditional-switch">conditional/switch</a> action to detect the data format
2. Then apply various actions to transform the value(s).
:::

## Transforming the data

1. First, detect the value using <a href="/docs/Documentation/actions-library/flow/conditional/action-conditional-switch">conditional/switch</a>.<br />
Set the <span class="action-field">SWITCH</span> condition to the input argument `source`
2. Second, apply <span class="action-field">CASE</span> where 'source' = 'website_user'
3. Third, for all other 'sources', apply a <span class="action-field">DEFAULT</span> case.

<img src={n_to_1_explanation} alt="N to 1 explanation" class="myResponsiveImg" width="800px"/>

## CASE: String split

In the first case, the name is joined in a single string.  Use [string/split](Documentation/actions-library/data/string/action-string-split.mdx) to split on every white space.

<img src={n_to_1_case} alt="N to 1 case" class="myResponsiveImg" width="800px"/>

## DEFAULT: Map the fields

Create a return object and map the values in the new JSON object.

<img src={n_to_1_return} alt="N to 1 return" class="myResponsiveImg" width="800px"/>

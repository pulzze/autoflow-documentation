# Workflow

Workflow consists of INPUT, Action, and OUTPUT.

✅ **INPUT Simulation** <br/>
Workflow receives the data through INPUT

✅ **Actions and Scope variables** <br/>
Performs some action(s)

✅ **OUTPUT Mapping** <br/>
Gives data back through OUTPUT

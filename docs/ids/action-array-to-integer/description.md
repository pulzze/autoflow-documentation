This Action returns Integer represented by the ordered.
*Each number must be a single digit.*
<br/>
- array : Target array
<br/>

**[Example]**
*Ex01)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
-----------------------
-> return : 123
```
*Ex02)*
```
[array]
  > number : 8
  > number : 7
  > number : 2
  > number : 0
-----------------------
-> return : 8720
```

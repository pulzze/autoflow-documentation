This Action returns result of Elixir command.
<br/>

- elixir : write elixir code.
- variables : want to write varibles

<br/>

**[Example]**
```
[elixir]
  > string : "ABC" <> "DEF" <> hello
[varialbes]
  > object
    > hello
      > string : GHI
------------------------------------
-> return : ABCDEFGHI
```

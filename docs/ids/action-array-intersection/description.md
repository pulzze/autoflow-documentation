This Action returns new Array containing only elements that 'array' and 'with' have in common.
- array : Target array.
- with : array to compare common parts
<br/>

**[Example]**
*Ex01)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
[with]
  > number : 2
  > number : 3
  > number : 4
-----------------------
-> return : [2, 3]
```

*Ex02)*
```
[array]
  > number : 1
  > number : 2
[with]
  > number : 3
  > number : 4
-----------------------
-> return : []
```

The This Action returns a string where all leading 'pattern' characters have been removed.

<br/>

- string : string value
- pattern : want to trim pattern

<br/>

**[Example]**
*Ex01)*
```
[string]
  > string : "__ abc _"
[pattern]
  > string : "_"
----------------------
-> return : " abc _"
```
*Ex02)*
```
[string]
  > string : "1 abc"
[pattern]
  > string : "11"
----------------------
-> return : "1 abc"
```

This Action returns the last element in Array or default if Array is empty.
- array : Target array

<br/>

**[Example]**
*Ex01)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
-----------------------
-> return : 3
```

*Ex02)*
```
[array]
  > number : 3
  > number : 5
-----------------------
-> return : 5
```

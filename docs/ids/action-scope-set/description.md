This Action returns value which you set in 'Scope' Value.
<br/>
- value : Set the value of the type you want.
<br/>

**[Example]**
*Ex01)*
```
[value] String > hello
----------------------
-> return : hello
```
*Ex02)*
```
[value] array
          > string : hello
          > string : autoflow
-----------------------------
-> return : ["hello", "autoflow"]
```

This Action returns true or false if string starts with any of the 'prefix' given.

<br/>

- string : string value
- prefix : want to check start of string value.

<br/>

**[Example]**
*Ex01)*
```
[string]
  > string : autoflow
[prefix]
  > string : auto
----------------------
-> return : true
```
*Ex02)*
```
[string]
  > string : autoflow
[prefix]
  > string : hola
----------------------
-> return : false
```

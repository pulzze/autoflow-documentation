This Action returns a string where converts all characters in the given string to uppercase.

<br/>

- string : string value

<br/>

**[Example]**
*Ex01)*
```
[string]
  > string : "hello"
----------------------
-> return : "HELLO"
```
*Ex02)*
```
[string]
  > string : "AutoFlow"
----------------------
-> return : "ATUOFLOW"
```

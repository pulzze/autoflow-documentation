Decodes a string encoded in specified base into a binary.

- base : Accepts an option to specify the base number from 2 to 36.

**[Example]**
*Ex01)*
```
[string]
  > string : 100
[base]
  > number : 8
-------------------------
-> return : @
```
*Ex02)*
```
[string]
  > string : 125
[base]
  > number : 16
-------------------------
-> return : %
```

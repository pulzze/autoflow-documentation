This action returns all keys in the object.
<br/>

- object : object value

<br/>

**[Example]**
*Ex01)*
```
[object]
  > object
      name : eric
      age : 22
------------------
-> return : ["name","age"]
```
*Ex02)*
```
[object]
  > object
      id : hello
      pw : qwer
      name : eric
      job : devloper
------------------
-> return : ["id","pw","name","job"]
```

This Action returns the minimal and the maximal element in Array.
- array : Target array

<br/>

**[Example]**
*Ex01)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
-----------------------
-> return : [1,3]
```

*Ex02)*
```
[array]
  > number : 3
  > number : 5
  > number : 11
  > number : 2
-----------------------
-> return : [2, 11]
```

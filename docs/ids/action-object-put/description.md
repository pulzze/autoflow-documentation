This action returns new object which puts the given 'value' under 'key' in 'object'
<br/>

- object : object value
- key : want to write new key
- value : want to wirte new value of the new key.

<br/>

**[Example]**
```
[object]
  > object
      id : 115
      pw : qwer
[key]
  > string : name
[value]
  > string : eric
----------------------
-> return : {"id": 115,"name":"eric","pw":"qwer"}
```

The This Action returns array that splits a given string based on a newline.

<br/>

- string : string value

<br/>

**[Example]**
*Ex01)*
```
[string]
  > string : I am Autoflow
             Thank you for using me.
------------------------------------
-> return : ["I am Autoflow", "Thank you for using me"]
```
*Ex02)*
```
[string]
  > string : nice
             meet
             you
-----------------------
-> return : ["nice", "meet", "you"]
```

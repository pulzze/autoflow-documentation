This Action returns array that splits a given string based on a newline.

- string : `string value`

### Examples

#### <Example 1>

**string**
```
I am Autoflow
Thank you for using me.
```

**returns**
```json
["I am Autoflow", "Thank you for using me"]
```


#### <Example 2>

**string**
```
nice
meet
you
```

**returns**
```json
["nice", "meet", "you"]
```

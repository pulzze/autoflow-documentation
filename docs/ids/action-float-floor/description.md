This Action return integer that rounds 'float' to the largest number less than or equal to 'float'.

<br/>

**[Example]**
*Ex01)*
```
[float] : 10.53
[precision] : 1
-------------
-> return : 10.5
```
*Ex02)*
```
[float] : 10.53
[precision] : 2
-------------
-> return : 10.52
```
*Ex03)*
```
[float] : 10.53
[precision] : 0
-------------
-> return : 10
```

This Action return integer that rounds 'float' to the smallest integer greater than or equal to 'float'.

<br/>

**[Example]**
*Ex01)*
```
[float] : 10.53
[precision] : 1
-------------
-> return : 10.6
```
*Ex02)*
```
[float] : 10.53
[precision] : 2
-------------
-> return : 10.53
```
*Ex03)*
```
[float] : 10.53
[precision] : 0
-------------
-> return : 11
```

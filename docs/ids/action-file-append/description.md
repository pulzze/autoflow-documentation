  File Append action appends content to a file, returns success
  If the file is not exist, the file will be created
  default path : `api-interactor/apps/common/data/files/`

  <br/>

  **[Example]**

  [input]
  ```
  { "content": "file test", "file": "test.txt" }
  ```

  [output]
  ```
  "success"
  ```

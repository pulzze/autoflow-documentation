This Action takes two arrays and merges them into one array.
<br/>
- array-left : Array to merge to the left.
- array-right : Array to merge to the right.

<br/>

**[Example]**

```
[array-left]
  > number : 1
  > number : 2
[array-right]
  > string : hello
  > string : autoflow
---------------------
-> return : [1,2,"hello","autoflow"]
```

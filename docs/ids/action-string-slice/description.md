This Action returns a substring starting at the offset 'start', and of the given 'length'.
If the offset is greater than string length, then it returns "".
<br/>

- string : string value
- start : start index of string
- lengh : Want to slice length

<br/>

**[Example]**
```
[string]
  > string : autoflow
[start]
  > number : 4
[length]
  > number : 4
-----------------------
-> return : flow
```

This Action returns the first element in Array or default if Array is empty.
- array : Target array

<br/>

**[Example]**
*Ex01)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
-----------------------
-> return : 1
```

*Ex02)*
```
[array]
  > number : 2
  > number : 3
-----------------------
-> return : 2
```

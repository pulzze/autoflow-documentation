This Action returns true or false if element exists within Array or not.
- array : Target array.
- element : value to check if it exists within Array.
<br/>

**[Example]**
*Ex01)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
[element]
  > number : 3
-----------------------
-> return : true
```

*Ex02)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
[element]
  > number : 5
-----------------------
-> return : false
```

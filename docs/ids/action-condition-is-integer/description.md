This Action returns true or false if the given input is integer type or not.

<br/>

**[Example]**
*Ex01)*
```
[value] : number > 10
--------------------
-> return : true
```
*Ex02)*
```
[value] : number > 10.1
--------------------
-> return : false
```
*Ex03)*
```
[value] : string > 10
--------------------
-> return : false
```

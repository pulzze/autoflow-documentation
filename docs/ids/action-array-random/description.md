This action returns random element in Array.
- array : Target array

<br/>

**[Example]**
*Ex01)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
  > string : hello
-----------------------
-> return : 1
```
*Ex02)*
```
[array]
  > number : 1
  > number : 2
  > number : 3
  > string : hello
-----------------------
-> return : hello
```

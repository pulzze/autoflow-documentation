Handles mixed case base 32 encoded string with extended hexadecimal alphabet

<br/>

**[Example]**
*Ex01)*
```
[string]
  > string : "CPNMUOJ1E8======"
-------------------------
-> return : "foobar"
```
*Ex02)*
```
[string]
  > string : "cpnMuOJ1E8======"
-------------------------
-> return : "foobar"
```
